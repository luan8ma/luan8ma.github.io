Welcome to Hexo! This is your very first post. Check documentation for more info. If you get any problems when using Hexo, you can find the answer in troubleshooting or you can ask me on GitHub.

  • Create a new post  $ hexo n "My New Post"
    More info: Writing

  • Run server  $ hexo s
    More info: Server

  • Generate static files  $ hexo g
    More info: Generating

  • Deploy to remote sites  $ hexo d
    More info: Deployment

    + hexo Nunjucks Errors hexo采用Nunjucks渲染文章可能将文章内容识别为其语法比如`{{}}`，然后报错。根据报错信息找到出错的地方，以 `{% raw %}出错内容{% endraw %}` 的方式包裹住出错内容，在markdown预览上会显示`{% raw %}`与`{% endraw %}`，实际渲染后会消失，且解决问题。
